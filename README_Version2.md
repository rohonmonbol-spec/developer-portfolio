# Abdul Rahaman Mondal - Developer Portfolio

A professional, animated portfolio website showcasing my skills, projects, and experience as a Full Stack Developer.

## 🌟 Features

- **Modern Design**: Dark theme with neon cyan accent colors and smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Advanced CSS animations including blob backgrounds, fade-ins, and hover effects
- **Professional Sections**:
  - Hero section with call-to-action buttons
  - About section with statistics
  - Skills section with progress bars (Frontend, Backend, Tools)
  - Featured projects showcase
  - Contact section with form and social links
  - Smooth navigation with active link highlighting

## 📱 Sections Included

1. **Home/Hero** - Introduction and main call-to-action

3. **Skills** - Technical skills with progress indicators
